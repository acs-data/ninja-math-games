# Ninja Math Games

Bienvenue dans **Ninja Math Games**, une application/interactivité amusante pour maîtriser les mathématiques comme un maître ninja ! 🥷

## 🚀 Fonctionnalités

- Sélectionnez votre **univers** et votre **héros**
- Répondez à des **questions interactives** qui couvrent différents sujets mathématiques (fractions, géométrie, calculs, etc.)
- **Débloquez des niveaux**, gagnez de l'XP et augmentez votre rang ninja
- Défis spéciaux comme le mode `Chronométré` ou les missions interactives de type **puzzle**
